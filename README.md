# python-core-to-ai

Structured learning repository: Python fundamentals → problem solving (LeetCode/HackerRank style) → mini-projects on the path toward AI/Data Engineering.

## Structure
- `core_python/` – core Python concepts, notes, and bug patterns
- `problems/` – coding practice (LeetCode/HackerRank-style)
- `mini_projects/` – small practical projects related to AI/Data workflows (text tools, mini ETL, etc.)
